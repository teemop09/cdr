# cdr
Windows cd command, but resolves symlinks/junctions to their target directory.

[c]hange [d]irectory [r]esolver

# Usage
Both `.bat` wrapper and `_cdr.exe` must be in directory listed in PATH.

Calling `cdr <path>` is equivalent to `cdr.bat <path>` and the batch wrapper script would invoke `_cdr.exe` or `pushd` as needed.

# Build
```
gcc cdr.c -o _cdr.exe
```
